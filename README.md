# 💡 Mini LED Moving Head

> Ein DIY-Projekt für einen selbst gebauten Mini-LED-Moving-Head auf Basis von Microcontrollern.

---

## 📋 Hardware

| Komponente | Modell |
|---|---|
| Microcontroller | ESP32-C3 Super Mini |
| LED | WS2812B RGBW (1 Pixel) |
| Servo Pan | SG90 |
| Servo Tilt | SG90 |

---
