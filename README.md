# 💡 Mini LED Moving Head

> A DIY mini LED moving head controller based on the ESP32-C3 microcontroller.

---

## 📋 Hardware

| Component | Model |
|---|---|
| Microcontroller | ESP32-C3 Super Mini |
| LED | WS2812B RGBW (1 pixel) |
| Pan servo | SG90 |
| Tilt servo | SG90 |

---
